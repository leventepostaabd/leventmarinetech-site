# /services Sinematik Sahne — foto klasörü

Buraya konan görseller, **/services** sayfasının SAĞ tarafındaki büyük "sahne"de,
sol listeden seçilen (veya kendiliğinden akan) sisteme göre gösterilir.

- **Yer:** `public/services/stage/`
- **Dosya adı = sistem slug'ı** (aşağıdaki liste). Uzantı: `.webp` (tercih) veya `.jpg`.
- **Boyut:** 1:1 kare, **1600×1600**, konu ortada + ~%15 kenar payı.
- Bir dosyayı koyduğun an sayfa **otomatik** kullanır (kod değişikliği yok).
  Dosya yoksa: önce eski broşür görseli, o da yoksa soft gradient + isim gösterilir.
- Promptlar için sohbetteki **MASTER (v2 soft & minimalist)** bloğunu kullan.
