# /credentials — sertifika küçük görselleri

About sayfasının altındaki **sertifika şeridinde** her belgenin küçük bir
**temsilî görseli** gösterilir (gerçek/kişisel tarama DEĞİL — o tip belgeye
benzeyen örnek). Görsel yoksa otomatik jenerik "belge" çizimi gösterilir.

- **Yer:** `public/credentials/`
- **Dosya adı = aşağıdaki tam adlar** (CertBadges.tsx ile eşleşir). Uzantı `.jpg`.
- **Boyut:** küçük (şeritte ~58×44 px gösterilir) — kaynak ~600×450 yeterli,
  hedef **<120 KB**. Kişisel/okunur bilgi olmasın (bu ölçekte okunmaz zaten).
- Tıklanmaz, büyümez — sadece "belge orada" hissi.
