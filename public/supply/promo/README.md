# /supply sol vitrin — promo görselleri

Buraya konan görseller, **/supply** sayfasının SOL panelinde sırayla (yumuşak
cross-fade) gösterilir — reklam/vitrin amaçlı: hizmet verdiğimiz limanlar, yeni
ürünler, akıllı denizcilik ürünleri, kendi markamız.

- **Yer:** `public/supply/promo/`
- **Dosya adı = slug** (aşağıda), uzantı `.webp` (tercih) / `.png` / `.jpg`.
- **Boyut:** 2:3 dikey, **1200×1800** (sol panel dar/portre), konu ortada.
- Servis tarafıyla aynı **soft & minimalist tema** (pale blue / off-white + amber
  vurgu) — bütünlük için sohbetteki MASTER v2 paletini kullan.
- Koyduğun an otomatik kullanılır; yoksa soft gradient + isim gösterilir.
- Slide listesini/başlıklarını `app/supply/page.tsx` içindeki `PROMO` dizisinden
  düzenleyebilirsin (isim, link, ekleme/çıkarma).
