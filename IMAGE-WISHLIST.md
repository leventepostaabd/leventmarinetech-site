# Image Wishlist — Levent Marine

> **Purpose.** Generate these images in Gemini (or any other image
> generator), download with the **exact filename** in the "Save as"
> column, then drop the files into the **exact path** shown. The site
> picks them up automatically — no code change needed.
>
> **Style guide for every prompt.** Photorealistic, cinematic lighting,
> documentary feel, **navy + amber palette** in the scene where possible
> (cool maritime steel tones, ambient warm work-light or amber control
> panels). Resolution **1600×1200** unless noted. JPG, q85, sRGB.
>
> Photo content rule: real marine work, never lifestyle stock. No
> people facing the camera (we want them at work). No watermarks, no
> dummy logos, no captions in the photo.

## 16 · Drop-in instructions

1. Open Gemini (or Midjourney, etc.).
2. Paste the prompt **exactly**.
3. Generate 4 variations; pick the best.
4. Download as JPG and rename to the **Save as** filename above.
5. Drop the file into the **Place at** path.
6. Commit + push. Site picks it up next deploy.

If a path doesn't exist yet, create the parent folder.

For images you can't use (poor result, anatomically odd), just don't
upload — the system gracefully skips missing files in the deck.
